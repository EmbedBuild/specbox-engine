# Contributing to SpecBox Engine

Thank you for your interest in SpecBox Engine. This is a proprietary project with controlled access.

## Rules

1. **No direct pushes to `main`** — All changes must go through a Pull Request.
2. **Review required** — Every PR requires approval from the project owner (@jesusperezdeveloper).
3. **One feature per PR** — Keep PRs focused and small.
4. **No force pushes** — Linear history is enforced; rebase your branch before opening the PR.
5. **Resolve conversations** — All review comments must be resolved before merge.

## Protected files

The following files require explicit owner approval and should not be modified without prior discussion:

- `ENGINE_VERSION.yaml`
- `CLAUDE.md`
- `install.sh`
- `.claude/settings.json`
- `rules/GLOBAL_RULES.md`
- `server/server.py`

## How to contribute

1. Create a feature branch from `main`
2. Make your changes
3. Open a Pull Request using the provided template
4. Wait for review and address any feedback
